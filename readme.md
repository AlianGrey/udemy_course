# 📚 JavaScript Learning Playground

## 📌 Overview
This repository contains a collection of JavaScript exercises, experiments and mini-projects created during in-depth study of the language.

It covers core concepts, asynchronous behavior, algorithms, browser APIs and practical problem solving.

The project serves as a personal knowledge base and hands-on practice for mastering JavaScript fundamentals.

---

## 🧠 Topics Covered

### 🔹 Core JavaScript
- Variables and data types  
- Conditions and loops  
- Functions and closures  
- Context (`this`)  
- Constructors and prototypes  

### 🔹 Advanced Concepts
- Recursion  
- Generators  
- Error handling (`try...catch`)  
- Deep cloning and JSON  
- Function composition  

### 🔹 Asynchronous JavaScript
- Event loop  
- Microtasks & macrotasks  
- Promises  
- Fetch API  

### 🔹 Data Structures & Algorithms
- Arrays and array methods  
- Factorial  
- Fibonacci  
- Pangram check  
- Deep array traversal
